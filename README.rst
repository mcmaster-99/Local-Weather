===================
Local Weather App
===================

- What the application does:

Displays the current weather of the user's location and a 6-day forecast.


- Tools Used:
1. HTML
2. CSS
3. Javascript
4. jQuery
5. Bootstrap
6. Gulp
7. Git
8. Github
